# Atividade Termômetro(Lista 0) - 

## Questões implementadas

- **number.c** — Impressão de números de 1 a N  
- **factorial.c** — Cálculo de Fatorial  
- **sumarray.c** — Soma de números em um array  
- **temp.c** — Conversão de Temperatura  
- **string.c** — Contagem de caracteres em uma string  
- **verifprime.c** — Verificação de número primo  

## Como compilar e executar

**Requisitos: Compilador GCC instalado**

**1. Entre na pasta do repositório**

   cd caminho/para/o/repositório

**2. Compile as questões (um comando por vez):**

   gcc number.c -o number  
   gcc factorial.c -o factorial  
   gcc sumarray.c -o sumarray  
   gcc temp.c -o temp  
   gcc string.c -o string  
   gcc verifprime.c -o verifprime  

   **(Se estiver no Windows e quiser .exe visível:)**

   gcc number.c -o number.exe  
   gcc factorial.c -o factorial.exe  
   gcc sumarray.c -o sumarray.exe  
   gcc temp.c -o temp.exe  
   gcc string.c -o string.exe  
   gcc verifprime.c -o verifprime.exe  

**3. Execute a que quiser:**

   **Linux/macOS:**

   ./number  
   ./factorial  
   ./sumarray  
   ./temp  
   ./string  
   ./verifprime  

   **Windows:**

   number.exe  
   factorial.exe  
   sumarray.exe  
   temp.exe  
   string.exe  
   verifprime.exe
